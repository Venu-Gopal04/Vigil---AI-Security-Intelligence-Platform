🎯 The Problem

Every morning at **6:10 AM**, Maya (Operations Lead) walks into the ops room with less than 2 hours before the 8:00 AM leadership briefing. She faces:

- 500+ raw security log entries
- Multiple disconnected dashboards
- No clear story of what actually happened overnight
- Time pressure to brief the Site Head

What happened? What matters? What needs escalation?



The Solution — VIGIL

VIGIL is an **AI-first overnight security intelligence platform** that automatically investigates all overnight security events the moment the operator opens the app.

**From 90 minutes of manual investigation → 10 minutes of AI-assisted review.**

✨Features

🤖 AI Agent with Real Tool Calling
- Autonomous investigation using 5 specialized tools
- Correlates events across time and location
- Surfaces uncertainty honestly — never guesses without data

🗺️ Interactive Site Map
- Real-time event plotting on dark satellite map
- Animated drone patrol simulation
- Click any event to see full details

🔍 Investigation Terminal
- Live terminal showing every AI tool call
- Watch the agent think in real time
- Full transparency of AI reasoning

👤 Human Review Layer
- Maya approves, escalates, or dismisses each event
- Add notes and overrides before briefing
- Human always stays in control

📋 Morning Briefing Generator
- Structured 5-section professional report
- Generated by AI, reviewed by human
- Downloadable as text file

🌐 Real Government Data
- Integrated with UK Police API (data.police.uk)
- Real crime incidents mapped to security event categories
- Live weather data from Open-Meteo API

🏃 Getting Started

 Prerequisites
- Node.js 18+
- Groq API key (free at console.groq.com)

Installation

```bash
Clone the repository
git clone https://github.com/Venu-Gopal04/Vigil---AI-Security-Intelligence-Platform.git

Go into the project
cd Vigil---AI-Security-Intelligence-Platform

# Install dependencies
npm install

Create environment file
cp .env.example .env.local


Environment Variables

Create `.env.local` in the root directory:


GROQ_API_KEY=your_groq_api_key_here


Get your free API key at 👉 [console.groq.com](https://console.groq.com)

Run Development Server

npm run dev


Open [http://localhost:3000](http://localhost:3000)

🔍 The 5 AI Tools Explained

| Tool | What it does |
|---|---|
| `get_event_details` | Fetches full details of any security event |
| `get_badge_history` | Checks employee badge swipe history |
| `get_drone_report` | Gets drone patrol findings for any area |
| `check_weather_conditions` | Real weather to detect false alarms |
| `correlate_events` | Finds connections between multiple events |

🎨 Design Philosophy

- **Dark, professional UI** — built for ops rooms, not offices
- **Information density** — everything visible at a glance
- **Trust indicators** — AI shows its work, humans verify
- **Time pressure UX** — countdown to briefing always visible
